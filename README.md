# AimVision — AI Enemy Detection APK

تطبيق أندرويد يستخدم الذكاء الاصطناعي للكشف عن الأعداء في الألعاب عبر قراءة الشاشة مباشرة.

## كيفية بناء APK

### الطريقة 1: GitHub Actions (الأسهل - مجاني)

1. ارفع هذا المشروع على GitHub
2. اذهب إلى **Actions** → **Build APK** → **Run workflow**
3. انتظر 5 دقائق ثم حمّل APK من **Artifacts**

### الطريقة 2: Android Studio

1. افتح المجلد في Android Studio
2. Build → Generate Signed APK أو Debug APK مباشرة

## الصلاحيات المطلوبة

| الصلاحية | السبب |
|---|---|
| `SYSTEM_ALERT_WINDOW` | رسم المربعات فوق الألعاب |
| `FOREGROUND_SERVICE_MEDIA_PROJECTION` | قراءة الشاشة في الخلفية |

## كيفية الاستخدام

1. ثبّت APK على هاتفك
2. افتح التطبيق
3. اضغط **تشغيل الكشف**
4. امنح إذن **الطبقة العلوية** (Overlay)
5. امنح إذن **تسجيل الشاشة** (Screen Capture)
6. افتح لعبتك — ستظهر مربعات حمراء على الأعداء فوراً!

## الميزات

- كشف الأعداء بـ TensorFlow Lite (EfficientDet)
- مربعات حمراء بتصميم Gaming HUD
- دائرة تصويب مع خط تقاطع
- يعمل على جميع الألعاب
- خط وصل من المركز لأقرب عدو
- عداد الأعداء في الأعلى

## متطلبات

- Android 8.0+ (API 26)
- 2GB RAM أو أكثر
